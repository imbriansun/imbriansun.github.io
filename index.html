margin: auto;

元素居中：

css width: 600px; margin: 0 auto;

唯一的问题是，当浏览器窗口比元素的宽度还要窄时，浏览器会显示一个水平滚动条来容纳页面。 ---> 在这种情况下使用 max-width 替代 width 可以使浏览器更好地处理小窗口的情况。这点在移动设备上显得尤为重要

html <!DOCTYPE html> <html> <head> <style> div.ex1 { width:500px; margin: auto; border: 3px solid #73AD21; }

div.ex2 { max-width:500px; margin: auto; border: 3px solid #73AD21; } </style> </head> <body>

<div class="ex1">This div element has width: 500px;</div> <br>

<div class="ex2">This div element has max-width: 500px;</div>

<p><strong>Tip:</strong> Drag the browser window to smaller than 500px wide, to see the difference between the two divs!</p>

</body> </html>

盒模型

width仅仅能设置元素的内容大小并不可以控制margin、padding和content总和，因此往往元素显示出的实际大小大于width，这时采用box-size解决这个问题
